# AI伴侣调教工坊

专为乙女和RP爱好者打造的AI角色扮演提示词生成器。

## 项目简介

这是一个基于 Next.js 和 Tailwind CSS 开发的静态网站，允许用户通过点选菜单（如世界观、人设、关系动态）来一键生成高质量的AI角色扮演提示词。

## 技术栈

- **框架**: Next.js 15 (App Router)
- **样式**: Tailwind CSS
- **语言**: TypeScript
- **部署**: 静态导出，可部署到 Vercel 免费套餐

## 开发

安装依赖：
```bash
npm install
```

启动开发服务器：
```bash
npm run dev
```

构建生产版本：
```bash
npm run build
```

## 部署

项目配置为静态导出模式，构建后会生成 `out` 目录，可直接部署到：
- Vercel
- Netlify
- GitHub Pages
- 任何静态网站托管服务
